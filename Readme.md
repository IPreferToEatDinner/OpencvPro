# OpencvPro

这里是 武汉大学遥感信息工程学院 数字图像处理大实习 **个人代码** 存放区

## 主要功能
完成了以下几个功能

- 打开图像
- 展示图像
- 图像滤波
	1. 高通滤波
	2. 低通滤波
	3. 中值滤波
	4. 自定义卷积核滤波
- 灰度变换
- 直方图匹配
- 图像二值化
	1. 状态法
	2. 判断分析法
- 图像运动与缩放
- 色彩平衡

以上这些功能都已经稳定，可以放心使用

## 界面实现

使用稳定的 `while` 循环来保证界面的交互性，在实际使用时，提供合理的提示词高亮等功能

## 代码风格

因为作者对函数式编程有所研究，所以在开发图像运动模块的时候可以发现，调用代码极为方便，建议其他开发者在改进代码时也可以将其他模块的代码改进为函数式

## 如何运行
因为本存储库已经完成了大部分操作，所以不需要复杂配置，请遵循下面的步骤即可启动项目

- 克隆项目或下载 `zip` 后打开项目
- 找到 `main` 文件，并启动此文件，这一步将会在文件夹中生成一个 `debug` 文件夹
- 将老师提供的 `opencv_world450d.dll` 文件放在 `OpencvPro/x64/Debug` 文件夹之中
- 启动项目，即可正常运行
